# Magic Ball 
A fun future predictor that answers yes/no questions with random mystical replies.

---

## 1. Description
- Language: Python
- Input: Yes/no questions + quit command
- Output: Random magic ball prediction
- Replies: 20 different mystical answers

---

## 2. How It Works
1. User is welcomed to the Magic Ball 8 game
2. User asks any yes/no question
3. Magic Ball randomly picks from 20 mystical replies
4. Reply is printed to the user
5. Loop continues until user types "quit"

---

## 3. Input / Output
| Question | Magic Ball Reply |
|---|---|
| Will I pass my exam? | "It is certain." |
| Should I apply for this job? | "Outlook good." |
| Is it going to rain tomorrow? | "Reply hazy, try again." |
| Will I get rich? | "Don't count on it." |
| quit | "Bye!!" |
